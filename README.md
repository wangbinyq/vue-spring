# vue-motion

> A very naive Vue implemation of [react-motion#v0.1.0](https://github.com/chenglou/react-motion/tree/v0.1.0)

## Install

``` bash
// npm
npm install --save vue-motion

// yarn
yarn add vue-motion
```
